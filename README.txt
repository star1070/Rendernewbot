
# Pi Auto Withdrawal Bot - Optimized (Regenerated)

## Features:
- 50+ parallel transactions at once
- Auto retry on errors (3 attempts, random delay)
- Locked/unlocked Pi balance handling
- Mobile-friendly UI
- Ready for Replit (free hosting)

## Setup on Replit:
1. Go to https://replit.com
2. Import this ZIP
3. In Shell, run:
   ```
   npm install express stellar-sdk stellar-hd-wallet body-parser
   node backend/server.js
   ```
4. Open `frontend/index.html` and start sending Pi!
